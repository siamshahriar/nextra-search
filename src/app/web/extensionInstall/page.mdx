# Extension Download and Installation

## Downloading the Extension

1. Log in to your QA Clan account at [https://app.qaclan.com/](https://app.qaclan.com/).
2. Navigate to the **Test Cases** section in the left-hand menu.

   ![Test Cases Menu](/images/web/projects.png)

3. If you're a new user, create a new project first, then go to **Test Cases**.
4. Click the **Extension** button to download the QAClan Test Case Recorder extension as a ZIP file.

   ![Download Extension](/images/web/extension.png)

---

## Installing the Extension in Chrome

Follow these steps to install the extension:

1. Extract the downloaded ZIP file to a folder on your computer.
2. Open Chrome and go to the extension management page by entering `chrome://extensions/` in the address bar.
3. Enable **Developer mode** by toggling the switch in the top-right corner.
4. Click **Load unpacked** and select the folder containing the extracted extension files.
5. Pin the QAClan Test Case Recorder extension for quick access by clicking the **extensions icon** and choosing the **pin option**.
