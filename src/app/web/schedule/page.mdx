# Schedule Feature Guide

The **Schedule** feature in QAClan allows users to run test cases periodically at predefined intervals. This helps automate test execution without manual intervention, ensuring continuous testing.

---

## Steps to Create a Schedule

1. Click **Schedule** in the left-side menu.

2. Click the **Schedule Test Case** button.

3. A menu will pop upâ€”select the desired test case from the dropdown menu.

4. Enter a **Cron Schedule Expression** in the provided field to define the execution timing.

5. Click the **Schedule Test Case** button to finalize and activate the schedule.
